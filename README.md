## Getting Started

### Prerequisites
Python 3.11.9

### Installation

1. Clone the Repository:
   ```sh
   git clone https://github.com/CicaMatt/PromptData.git
   ```
2. Install Dependencies:
   ```sh
   pip install pandas
   pip install scipy
   ```
